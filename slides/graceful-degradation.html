Graceful Degradation

Profiter des nouvelles fonctionnalités et ensuite prévoir un moyen de fonctionner sur les systèmes aux capacités limitées.

  • Les technologies Web HTML/CSS sont naturellement prévues pour fonctionner dans des plateformes ne supportant pas toutes les fonctionnalités
  • Si un tag HTML ou une propriété CSS ne sont pas supportées --> non prise en compte de manière silencieuse
